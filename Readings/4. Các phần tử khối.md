# 4. Các phần tử khối

## 4.1. Trích dẫn - Blockquote

Thêm dấu `>` vào trước mỗi dòng trích dẫn.

Cú pháp:

```markdown
> Trích dẫn 1
> Trích dẫn 2

. . .
```

Kết quả:

> Trích dẫn 1

> Trích dẫn 2

. . .

---

## 4.2 Code trong lòng - Inline Code

Để viết **inline <code>**, dùng 2 dấu ` ở trước và sau từ đó.

Cú pháp:

```markdown
`inline code`
```

Kết quả:

`inline code`

---

## 4.3. Khối lệnh - Block Code

Để viết một đoạn `<code>`, ta chỉ cần dùng 3 dấu ngoặc kép ngược ` ở trước và sau đó có thể thêm format ngôn ngữ lập trình đó để tô sáng cú pháp (syntax highlighting).

Cú pháp:

````markdown
```python
print("hello world")
```
````

Kết quả:

```python
print("hello world")
```

Để viết được cú pháp chứa lệnh python trên làm ví dụ mà không bị ảnh hướng bởi ba dấu ngoặc kép ngược ta cần bổ sung thêm một dấu ngoặc kép ngược nữa vào trong ba dấu đó, tạo thành bốn dấu ngoặc kép ngược.
